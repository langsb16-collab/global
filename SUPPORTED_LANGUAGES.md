## 🚀 사용 방법

### 언어 선택하기

1. **드롭다운 메뉴**: 페이지 상단 언어 선택기 클릭
2. **국기 아이콘**: 시각적으로 쉬운 언어 식별
3. **자동 저장**: LocalStorage에 선택 저장
4. **즉시 적용**: 페이지 리로드로 언어 변경

### 기본 언어 설정

```javascript
// 브라우저 언어 감지 (계획 중)
const browserLang = navigator.language.split('-')[0];

// 수동 언어 설정
localStorage.setItem('farm2world_lang', 'en-CA');
```
